# CS-Ebook 网站设计方案

## 选定设计方案：深色极简科技风格

### 设计哲学
**极简科技主义** - 以深色为主调，融合霓虹绿色的科技感，营造现代、专业、沉浸式的阅读体验。强调内容本身，通过极简的界面设计让用户专注于书籍浏览。

### 核心原则
1. **内容至上** - 书籍信息是主角，界面设计退居其后，不抢夺用户注意力
2. **科技感与可读性** - 使用霓虹绿色作为强调色，但保持足够的对比度确保易读性
3. **深色沉浸** - 深色背景减少眼睛疲劳，适合长时间浏览
4. **清晰的信息层级** - 通过间距、大小、颜色区分不同信息优先级

### 色彩哲学
- **主背景**：纯黑色（#000000）或接近黑色的深灰（#0a0a0a）- 营造深邃、专业的氛围
- **次背景**：石板灰（#1a1a1a, #2d2d2d）- 用于卡片、容器，提供层次感
- **强调色**：霓虹绿色（#00ff88, #00dd77）- 用于交互元素、链接、重点信息
- **文本色**：浅灰白（#e8e8e8, #f0f0f0）- 确保深色背景上的可读性
- **边框/分割线**：深灰（#333333, #404040）- 微妙的分割，不显得生硬

### 布局范式
- **非对称网格布局** - 避免居中单调的设计
- **左侧导航/分类** + **右侧内容区** 的经典两栏结构
- **卡片网格** - 书籍展示使用响应式卡片网格，3-5 列布局
- **粘性导航** - 顶部导航栏固定，方便快速跳转

### 签名元素
1. **霓虹绿色边框/下划线** - 用于活跃导航项、悬停效果
2. **渐进式阴影** - 深色背景上使用微妙的阴影营造深度感
3. **代码风格的排版** - 某些标题使用等宽字体或特殊排版，强化科技感

### 交互哲学
- **平滑过渡** - 所有交互都有 200-300ms 的过渡动画
- **霓虹绿色反馈** - 悬停、点击时出现绿色高亮或边框
- **微妙动画** - 卡片进入时的淡入、滑入效果，不过度
- **即时反馈** - 搜索、筛选时立即更新结果

### 动画指南
- **进入动画**：淡入 + 微妙上滑（200ms）
- **悬停效果**：背景色变化、绿色边框出现（150ms）
- **过渡动画**：页面切换、分类展开时的平滑过渡（300ms）
- **加载状态**：绿色脉冲或旋转动画，强化科技感

### 排版系统
- **标题字体**：Courier Prime / IBM Plex Mono（等宽字体，科技感）
  - H1: 48px, 700 weight
  - H2: 32px, 600 weight
  - H3: 24px, 600 weight
- **正文字体**：Segoe UI / -apple-system（系统字体，高可读性）
  - 正文: 16px, 400 weight
  - 小文本: 14px, 400 weight
- **代码/标签**：Fira Code（等宽，用于分类标签）
  - 12-14px, 500 weight

## 设计决策记录

### 为什么选择这个方案？
- **深色主题** 符合现代开发者审美，适合长时间阅读技术文档
- **霓虹绿色** 是经典的黑客/科技审美，与计算机科学主题相得益彰
- **极简设计** 不分散注意力，让书籍内容成为焦点
- **非对称布局** 避免了常见的居中单调设计，更显专业

### 实现优先级
1. 基础深色主题 + 霓虹绿色强调色
2. 导航和分类系统
3. 书籍卡片网格展示
4. 搜索和筛选功能
5. 细节动画和交互优化
