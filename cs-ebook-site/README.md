# CS-Ebook Explorer

一个简约、深色风格的计算机科学电子书浏览网站，基于 [HiMenma/CS-Ebook](https://github.com/HiMenma/CS-Ebook) GitHub 项目。

## 设计特点

### 视觉风格

- **深色极简科技主义**：纯黑色背景（#000000）营造深邃、专业的氛围
- **霓虹绿色强调**：#00ff88 作为主要强调色，提供现代科技感
- **石板灰辅助**：#2d2d2d 和 #404040 用于卡片和容器，增加层次感

### 核心原则

1. **内容至上** - 书籍信息是主角，界面设计退居其后
2. **科技感与可读性** - 霓虹绿色强调但保持足够对比度
3. **深色沉浸** - 减少眼睛疲劳，适合长时间浏览
4. **清晰的信息层级** - 通过间距、大小、颜色区分优先级

## 功能特性

### 浏览功能

- **分类导航** - 左侧侧栏展示所有书籍分类和子分类
- **快速搜索** - 实时搜索书籍名称、作者或分类
- **网格展示** - 响应式书籍卡片网格，支持多种屏幕尺寸
- **交互反馈** - 悬停效果、平滑过渡、霓虹绿色高亮

## 技术栈

- **前端框架**：React 19 + TypeScript
- **样式系统**：Tailwind CSS 4 + 自定义深色主题
- **路由**：Wouter（轻量级客户端路由）
- **UI 组件**：shadcn/ui（现代化组件库）
- **图标**：Lucide React
- **构建工具**：Vite

## 快速开始

### 开发模式

```bash
cd /home/ubuntu/cs-ebook-site
pnpm install
pnpm dev
```

开发服务器将在 `http://localhost:3000` 启动。

### 构建生产版本

```bash
pnpm build
```

### 预览生产构建

```bash
pnpm preview
```

## 设计细节

### 色彩系统

- **背景色**：#000000（纯黑）
- **卡片色**：#0a0a0a（深灰）
- **文本色**：#e8e8e8（浅灰白）
- **强调色**：#00ff88（霓虹绿）
- **边框色**：#333333（深灰）

### 排版

- **标题字体**：Courier Prime, JetBrains Mono（等宽字体，科技感）
- **正文字体**：系统字体栈（高可读性）
- **字体大小**：H1 48px, H2 32px, H3 24px, 正文 16px

### 交互动画

- **进入动画**：淡入 + 微妙上滑（200ms）
- **悬停效果**：背景色变化、绿色边框出现（150ms）
- **过渡动画**：平滑过渡（300ms）

## 使用指南

### 浏览书籍

1. 在左侧侧栏选择分类，自动展开子分类
2. 点击子分类查看该类别的所有书籍
3. 书籍以网格形式展示，支持响应式布局

### 搜索书籍

1. 在顶部搜索栏输入书籍名称、作者或分类
2. 搜索结果实时更新
3. 点击清除按钮（X）清空搜索

### 书籍信息

- 每本书籍显示书名和所属分类
- 悬停卡片显示交互反馈（边框高亮、阴影增强）
